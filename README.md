Number Guessing Game (Python)

A simple command-line game built using Python where the user tries to guess a randomly generated number.

Features

Generates a random number between 1 and 100
Takes user input
Provides feedback:
  Too high
  Too low
  Correct guess
Tracks number of attempts

Tech Stack
  Python 3
  Built-in module:
    random

Project Structure
  number-guessing-game/
  │
  ├── app.py        # Main game logic
  └── README.md     # Documentation
  

How It Works

The computer generates a random number between 1 and 100
You enter your guess
The program tells you if:
Your guess is too small
Your guess is too large
You guessed correctly
It shows the number of attempts
